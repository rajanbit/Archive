Tool: TYGS [Type (Strain) Genome Server]
Perform: Genome-based prokaryote Taxonomy

>Website
https://tygs.dsmz.de/
https://tygs.dsmz.de/user_requests/new

>Usage
Go to @ https://tygs.dsmz.de/user_requests/new
Browse and select all the Genomic Fasta files
Enter the mail id
Submit the job

>Result
GBDP tree (16S rDNA gene sequence-based)
GBDP tree (whole-genome sequence-based)
Identification Table containing name of species the sequences belong to
Pairwise comparisons of user genomes vs. type strain genomes
Strains in your dataset
