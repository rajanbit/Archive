Tool: Vim
Perform: Text editor for terminal

Website: https://www.vim.org/

>Installation
$ sudo apt-get update
$ sudo apt-get install vim

>Usage

Create a file
$ vim <file_name>

Open and Edit a file
$ vim <file_name>
$ a [append]
